# Kyle Salomon - Personal Website

A modern, professional personal website showcasing my experience as an Analytics Leader and Data Professional.

## Project Structure

```
personal-website/
├── index.html          # Main HTML file
├── styles.css          # CSS styling
├── script.js           # JavaScript for interactivity
└── README.md          # This file
```

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Easy Navigation**: Fixed navigation bar with smooth scrolling
- **Sections**:
  - Hero section with quick contact options
  - About Me with personal interests
  - Professional Experience timeline
  - Skills showcase
  - Education & Certifications
  - Contact information

## How to Use

### View Locally

1. Open `index.html` in any modern web browser
2. That's it! No build process or server required.

### Deployment Options

You can deploy this website to:

1. **GitHub Pages** (Free)
   - Create a GitHub repository
   - Push your code
   - Enable GitHub Pages in repository settings

2. **Netlify** (Free)
   - Drag and drop the folder to Netlify
   - Get instant deployment with custom domain support

3. **Vercel** (Free)
   - Import your GitHub repository
   - Automatic deployments on every push

4. **Your Own Hosting**
   - Upload files via FTP to any web hosting service

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    /* etc. */
}
```

### Content
All content is in `index.html`. Simply edit the text within the HTML tags.

### Adding New Sections
1. Add a new `<section>` in `index.html`
2. Add a navigation link in the navbar
3. Style it in `styles.css`
